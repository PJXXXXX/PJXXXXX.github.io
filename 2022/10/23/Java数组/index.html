Java数组

发表于2022-10-23|更新于2022-10-24|Java数组
|阅读量:

数组初始化

  • 动态初始化 ：

    • 数据类型 数组名[] = new 数据类型[大小] , int a[] = new int[5]
    • 先声明，后创建 。int[] a; a= new int[n]

  • 静态初始化：

    • int a[] = {1,2,3,4}

  • 默认值：int 0; short 0;byte 0;long 0;float 0.0;double 0.0;char \u0000;boolean false;String null

数组赋值机制

  • 基本数据类型复制，赋的是具体数据，值拷贝，相互不影响

  • 数组在默认情况下是引用传递，赋的是地址，地址拷贝

    • int[] arr1 = {1,2,3};
int[] arr2 = arr1;
arr2[0] = 10;  //arr1={10,2,3},arr2={10,2,3}
/*
jvm的内存：栈 堆 方法区
基本数据类型：直接放在栈
引用数据类型：变量在栈中指向堆中存放数据的地址。 地址拷贝，指向同一个堆中数据空间
字符串：存放在常量池中  jdk1.6：常量池存放在方法区；jdk1.7: 常量池存放在堆中
类s信息加载到方法区，属性信息，行为（方法）信息
*/

二维数组

  • 定义：int[][] y；int[] y[]；int y[][]
  • 各个一维数组长度可以不相同 如：int map[][] = { {1，2}，{3，4，5} }；
